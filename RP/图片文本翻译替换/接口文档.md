# 图片文本翻译替换 - 接口文档（V1.0）

本应用只有 **1个接口**。

**批量处理说明：** 接口每次处理1张图片，批量上传时前端逐张调用此接口，最多3张并发。服务端无需感知"批量"概念，每次请求独立处理即可。

---

## POST /api/translate

**说明：** 接收图片 base64，调用魔塔大模型识别中文文本并翻译，返回替换文字后的新图片。

### 请求

`Content-Type: application/json`

```json
{
  "id": "abc123",
  "base64": "iVBORw0KGgoAAAANSUhEUgAA...",
  "format": "jpeg",
  "targetLang": "en"
}
```

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| id | string | 是 | 前端生成的临时 ID，用于匹配响应 |
| base64 | string | 是 | 图片 base64（不含 `data:image/xxx;base64,` 前缀） |
| format | string | 是 | 图片格式：`jpeg` / `png` / `webp` |
| targetLang | string | 是 | 目标语言：`en` `ja` `ko` `es` `fr` `de` `ru` `th` `vi` `id` |

### 响应 - 成功

```json
{
  "id": "abc123",
  "success": true,
  "translatedBase64": "iVBORw0KGgoAAAANSUhEUgAA...",
  "texts": [
    { "original": "立即购买", "translated": "Buy Now" },
    { "original": "限时特惠", "translated": "Limited Time Offer" }
  ]
}
```

### 响应 - 失败

```json
{
  "id": "abc123",
  "success": false,
  "error": "未识别到中文文本"
}
```

| error 内容 | 原因 |
| --- | --- |
| 未识别到中文文本 | 图片中无中文，或图片模糊 |
| 缺少必要参数 | 请求体缺少字段 |
| 处理失败 | 服务端异常 |

### 前端调用示例

```typescript
const res = await fetch('/api/translate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ id, base64, format, targetLang })
});
const data = await res.json();

if (data.success) {
  // 使用处理后的图片
  const imgSrc = `data:image/jpeg;base64,${data.translatedBase64}`;
}
```

### 处理时序

```
前端 → POST /api/translate
         ↓
    qwen-vl-plus（识别中文文本及位置）
         ↓
    qwen-turbo（翻译文本）
         ↓
    Sharp + Canvas（替换图片文字）
         ↓
前端 ← 返回 translatedBase64
```

### 限流

每 IP 每分钟最多 20 次请求，超出返回 HTTP 429。
