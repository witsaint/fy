# 后端需求文档（Next.js版本）

> **说明**：本项目使用 Next.js 全栈开发，API路由替代传统后端服务

## 一、技术栈

### 框架与语言
- **框架**：Next.js 14+ (App Router)
- **语言**：TypeScript
- **API实现**：Next.js API Routes
- **数据存储**：无数据库，内存临时存储

### 第三方服务
- **OCR服务**：百度OCR API（推荐，免费1000次/天）
- **AI解析**：魔塔（ModelScope）API - OpenAI兼容接口

## 二、API路由设计

> **说明**：采用Next.js API Routes，无数据库设计，数据在前端React State中管理

### 核心API：仅1个识别接口

由于无数据库设计，后端仅需提供**图片识别API**，其他操作（更新、删除、筛选）都在前端完成。

### 2.1 图片识别API（唯一后端接口）

#### 接口定义
```
POST /api/recognize

Request:
{
  "image": "data:image/jpeg;base64,/9j/4AAQSkZJRg..." // Base64编码的图片
}

Response:
{
  "success": true,
  "data": {
    "id": "prop-uuid-1",
    "region": "瑶海区",
    "building": "万事通大厦",
    "roomNumber": "1005",
    "area": "252㎡",
    "price": "42元/㎡/天",
    "propertyFee": "3.5元/㎡/月",
    "commission": "全佣金",
    "remarks": "含家具，电梯口",
    "confidence": 0.96,
    "createdAt": "2026-02-15T10:00:00.000Z"
  }
}

// 错误响应
{
  "success": false,
  "error": "OCR识别失败：图片模糊"
}
```

#### 功能要求

##### 1. OCR图像识别
- **服务选择**：
  - 百度OCR API（高精度版）
  - 腾讯OCR API（通用印刷体识别）
  - 阿里云OCR API
  - 开源方案：PaddleOCR（支持本地部署）
  
- **识别流程**：
  1. 读取图片文件
  2. 调用OCR API获取文字内容
  3. 返回原始文字和坐标信息

##### 2. AI结构化解析（核心难点）
- **采用方案**：魔塔（ModelScope）API - OpenAI兼容接口

**魔塔API配置**：
```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: process.env.MODELSCOPE_ACCESS_TOKEN || 'ms-571153f6-c26d-4dba-b4c2-ec6a88450519',
  baseURL: 'https://api-inference.modelscope.cn/v1/'
});
```

**推荐模型**：
- `qwen-turbo`: 速度快，成本低（推荐生产环境）
- `qwen-plus`: 平衡性能和成本
- `qwen-max`: 最高精度（复杂场景）

**解析Prompt示例**：
```typescript
const prompt = `你是房源信息提取助手。请从以下OCR识别的文本中，精准提取8个字段：

1. 区域：房源所属行政区域（瑶海区、经开区、蜀山区、庐阳区、包河区、新站区）
2. 楼盘：楼盘名称全称
3. 房号：房间编号（保留原格式）
4. 面积：面积+单位（如252㎡）
5. 价格：租金价格+单位（如42元/㎡/天）
6. 物业费：物业费+单位（如3.5元/㎡/月）
7. 佣金：佣金政策
8. 备注：关键补充信息

OCR识别结果：
${ocrText}

请以JSON格式返回（仅返回JSON，不要其他内容）：
{
  "region": "",
  "building": "",
  "roomNumber": "",
  "area": "",
  "price": "",
  "propertyFee": "",
  "commission": "",
  "remarks": ""
}`;

const completion = await client.chat.completions.create({
  model: 'qwen-turbo',
  messages: [
    { role: 'system', content: '你是一个专业的房源信息提取助手。' },
    { role: 'user', content: prompt }
  ],
  temperature: 0.1,
  max_tokens: 1024
});
```

**魔塔API优势**：
- ✅ OpenAI兼容接口，代码简洁
- ✅ 中文理解能力强
- ✅ 国内服务，访问速度快
- ✅ 免费额度充足
- ✅ 无需代理，稳定可靠

##### 3. 数据规整
- **区域标准化**：自动匹配6个标准区域名称
- **面积单位补充**：无单位时自动添加"㎡"
- **空值处理**：识别为空的字段标注"无"
- **格式统一**：去除多余空格、统一标点符号

##### 4. 识别置信度计算
- 核心字段（区域、楼盘、房号、价格）有值：置信度+0.6
- 其他字段有值：每个字段+0.1
- 低于0.8置信度的结果标记为"需人工检查"

#### 技术实现要点
1. **异步处理**：使用队列（如Bull/Redis Queue）处理批量识别
2. **进度反馈**：通过WebSocket实时推送识别进度
3. **错误重试**：OCR/AI调用失败时自动重试3次
4. **日志记录**：记录识别日志用于后续优化

```typescript

    p.id === id ? { ...p, ...updates } : p
  ));

```

##### 4. 数据删除（前端）
```typescript

  setProperties(prev => prev.filter(p => p.id !== id));

const clearAllData = () => {
  setProperties([]);
  setImages([]);
};
```

##### 5. Excel导出（前端生成）
```typescript
import * as XLSX from 'xlsx';

// 前端直接生成Excel文件
const exportToExcel = (properties: Property[]) => {
  const data = properties.map(p => ({
    '区域': p.region,
    '楼盘': p.building,
    '房号': p.roomNumber,
    '面积(㎡)': p.area,
    '价格': p.price,
    '物业费(元/㎡/月)': p.propertyFee,
    '佣金': p.commission,

  XLSX.utils.book_append_sheet(workbook, worksheet, '房源汇总');
  
  const date = new Date().toISOString().split('T')[0].replace(/-/g, '');
  XLSX.writeFile(workbook, `房源汇总表_${date}.xlsx`);
};
```

**优势**：

- ✅ 代码简单，易于维护

**特点**：

## 三、非功能需求

### 3.1 性能要求
- 单张图片OCR识别时间：≤5秒
- AI解析时间：≤3秒（魔塔API响应快）
- 总体识别时间：≤10秒/张
- 批量识别：前端控制并发数5-10，避免API限流
- Excel导出（前端生成）：≤2秒（100条数据）

### 3.2 安全性要求
1. **数据隔离**：数据仅存储在前端内存，不持久化
2. **API密钥保护**：密钥存储在环境变量，不提交到代码仓库
3. **输入验证**：前端验证图片格式和大小
4. **错误处理**：敏感信息不暴露到错误消息中

### 3.3 可靠性要求
1. **错误重试**：OCR/AI调用失败时前端自动重试2-3次
2. **降级策略**：识别失败时标记"识别失败"，允许手动编辑
3. **错误日志**：记录识别失败的图片和错误信息

### 3.4 可维护性要求
1. **配置外置**：API密钥配置在.env.local
2. **模块解耦**：OCR、AI解析、数据规整独立模块
3. **TypeScript类型**：完整的类型定义，代码提示完善
4. **代码注释**：核心函数添加注释说明

## 四、第三方服务集成

### 4.1 OCR服务：百度OCR

**选择理由**：
- ✅ 准确率高（通用文字识别高精度版）
- ✅ 免费额度充足（1000次/天）
- ✅ 接口稳定，响应快
- ✅ 支持多种图片格式

**API调用示例**：
```typescript
// lib/ocr.ts
import axios from 'axios';

export async function recognizeImage(base64Image: string): Promise<string> {
  // 1. 获取Access Token
  const tokenResponse = await axios.post(
    'https://aip.baidubce.com/oauth/2.0/token',
    null,
    {
      params: {
        grant_type: 'client_credentials',
        client_id: process.env.BAIDU_API_KEY,
        client_secret: process.env.BAIDU_SECRET_KEY
      }
    }
  );
  
  const accessToken = tokenResponse.data.access_token;
  
  // 2. 调用OCR识别
  const imageData = base64Image.includes(',') 
    ? base64Image.split(',')[1] 
    : base64Image;
  
  const ocrResponse = await axios.post(
    `https://aip.baidubce.com/rest/2.0/ocr/v1/accurate_basic?access_token=${accessToken}`,
    `image=${encodeURIComponent(imageData)}`,
    {
      headers: {
        'Content-Type': 'application/x-www-form-urlencoded'
      }
    }
  );
  
  // 3. 提取文字
  const words = ocrResponse.data.words_result || [];
  return words.map((item: any) => item.words).join('\n');
}
```
