## 1. 文件上传接口

### 基本信息
- **接口地址**：`POST /api/upload`
- **接口描述**：批量上传房源图片，支持最多20张图片同时上传
- **Content-Type**：`multipart/form-data`

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| files | File[] | 是 | 图片文件数组，最多20个 |

### 请求示例
```javascript
const formData = new FormData();
files.forEach(file => {
  formData.append('files', file);
});

fetch('http://localhost:3000/api/upload', {
  method: 'POST',
  body: formData
})
```

### 响应参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| success | Boolean | 请求是否成功 |
| message | String | 提示信息 |
| data | Object | 返回数据 |
| data.uploadedCount | Number | 成功上传的文件数量 |
| data.files | Array | 上传文件信息列表 |
| data.files[].id | String | 文件唯一ID |
| data.files[].originalName | String | 原始文件名 |
| data.files[].size | Number | 文件大小（字节） |
| data.files[].path | String | 文件存储路径 |
| data.files[].uploadTime | String | 上传时间 |

### 成功响应示例
```json
{
  "success": true,
  "message": "上传成功",
  "data": {
    "uploadedCount": 3,
    "files": [
      {
        "id": "uuid-001",
        "originalName": "房源1.jpg",
        "size": 2048576,
        "path": "/uploads/uuid-001.jpg",
        "uploadTime": "2026-02-15T10:30:00.000Z"
      },
      {
        "id": "uuid-002",
        "originalName": "房源2.png",
        "size": 1536000,
        "path": "/uploads/uuid-002.png",
        "uploadTime": "2026-02-15T10:30:01.000Z"
      }
    ]
  }
}
```

### 错误响应示例
```json
{
  "success": false,
  "message": "文件格式错误",
  "error": {
    "code": "INVALID_FILE_FORMAT",
    "details": "仅支持JPG/PNG格式，单张≤10MB"
  }
}
```

### 错误码说明
| 错误码 | 说明 |
|--------|------|
| INVALID_FILE_FORMAT | 文件格式不正确（非JPG/PNG） |
| FILE_TOO_LARGE | 文件大小超过10MB |
| TOO_MANY_FILES | 上传文件数量超过20个 |
| UPLOAD_FAILED | 上传失败（网络或服务器错误） |

## 2. 图片识别接口

### 基本信息
- **接口地址**：`POST /api/recognize`
- **接口描述**：对上传的图片进行OCR识别和AI解析，提取房源信息
- **Content-Type**：`application/json`

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileIds | String[] | 是 | 要识别的文件ID数组 |
| reRecognize | Boolean | 否 | 是否重新识别（默认false） |

### 请求示例
```json
{
  "fileIds": ["uuid-001", "uuid-002", "uuid-003"],
  "reRecognize": false
}
```

### 响应参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| success | Boolean | 请求是否成功 |
| message | String | 提示信息 |
| data | Object | 返回数据 |
| data.total | Number | 总共要识别的图片数量 |
| data.recognized | Number | 成功识别的数量 |
| data.failed | Number | 识别失败的数量 |
| data.results | Array | 识别结果列表 |
| data.errors | Array | 识别失败的文件信息 |

### 识别结果对象（results[i]）
| 参数名 | 类型 | 说明 |
|--------|------|------|
| fileId | String | 文件ID |
| propertyId | String | 生成的房源信息ID |
| region | String | 区域 |
| building | String | 楼盘 |
| roomNumber | String | 房号 |
| area | String | 面积 |
| price | String | 价格 |
| propertyFee | String | 物业费 |
| commission | String | 佣金 |
| remarks | String | 备注 |
| confidence | Number | 识别置信度（0-1） |

### 成功响应示例
```json
{
```

## 3. 获取房源列表接口

### 基本信息
- **接口地址**：`GET /api/properties`
- **接口描述**：获取房源信息列表，支持分页和筛选
- **Content-Type**：`application/json`

### 请求参数（Query参数）
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| region | String | 否 | 区域筛选（不传则获取全部） |
| page | Number | 否 | 页码（默认1） |
| limit | Number | 否 | 每页数量（默认20） |
| sortBy | String | 否 | 排序字段（默认created_at） |
| sortOrder | String | 否 | 排序方向：asc/desc（默认desc） |

### 请求示例
```
GET /api/properties?region=瑶海区&page=1&limit=20&sortBy=created_at&sortOrder=desc
```

### 响应参数
| 参数名 | 类型 | 说明 |
|--------|------|------|
| success | Boolean | 请求是否成功 |
| data | Object | 返回数据 |
| data.total | Number | 总记录数 |
| data.page | Number | 当前页码 |
| data.limit | Number | 每页数量 |
| data.totalPages | Number | 总页数 |
| data.items | Array | 房源信息列表 |

### 成功响应示例
```json
{
```

## 8. 导出Excel接口

### 基本信息
- **接口地址**：`POST /api/export/excel`
- **接口描述**：导出房源信息为Excel文件
- **Content-Type**：`application/json`

### 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| region | String | 否 | 区域筛选（不传则导出全部） |
| ids | String[] | 否 | 指定要导出的房源ID |

### 请求示例
```json
{
  "region": "瑶海区"
}
```

或指定ID导出：
```json
{
  "ids": ["prop-uuid-001", "prop-uuid-002"]
}
```

### 成功响应示例
```json
{
  "success": true,
  "message": "导出成功",
  "data": {
    "downloadUrl": "/downloads/房源汇总表_20260215.xlsx",
    "fileName": "房源汇总表_20260215.xlsx",
    "fileSize": 15360,
    "recordCount": 45
  }
}
```

### 前端下载处理
```javascript
const response = await fetch('/api/export/excel', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ region: '瑶海区' })
});

const result = await response.json();

// 触发下载
const link = document.createElement('a');
link.href = result.data.downloadUrl;
link.download = result.data.fileName;
link.click();
```

## 10. WebSocket实时进度

### 基本信息
- **连接地址**：`ws://localhost:3000/ws/progress`
- **协议**：WebSocket
- **用途**：实时接收图片识别进度

### 连接示例
```javascript
const ws = new WebSocket('ws://localhost:3000/ws/progress');

ws.onopen = () => {
  console.log('WebSocket连接已建立');
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('识别进度:', data);
  
  // 更新前端进度显示
  updateProgress(data);
};

ws.onerror = (error) => {
  console.error('WebSocket错误:', error);
};

ws.onclose = () => {
  console.log('WebSocket连接已关闭');
};
```

### 消息格式
```json
{
  "type": "progress",
  "data": {
    "current": 3,
    "total": 10,
    "percentage": 30,
    "currentFile": "房源3.jpg",
    "status": "recognizing",
    "message": "正在识别第3张图片"
  }
}
```

### 消息类型
| type值 | 说明 |
|--------|------|
| progress | 进度更新 |
| completed | 全部识别完成 |
| error | 识别出错 |

### 完成消息示例
```json
{
  "type": "completed",
  "data": {
    "total": 10,
    "recognized": 9,
    "failed": 1,
    "message": "识别完成，共提取9条房源信息"
  }
}
```

## 接口调用示例（完整流程）

### 1. 上传图片
```javascript
// 上传图片
const uploadFiles = async (files) => {
  const formData = new FormData();
  files.forEach(file => formData.append('files', file));
  
  const response = await fetch('/api/upload', {
    method: 'POST',
    body: formData
  });
  
  return await response.json();
};
```

### 2. 触发识别
```javascript
// 触发识别
const recognizeImages = async (fileIds) => {
  const response = await fetch('/api/recognize', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ fileIds })
  });
  
  return await response.json();
};
```

### 3. 获取列表
```javascript
// 获取房源列表
const getProperties = async (region, page = 1) => {
  const params = new URLSearchParams({ page, limit: 20 });
  if (region) params.append('region', region);
  
  const response = await fetch(`/api/properties?${params}`);
  return await response.json();
};
```

### 4. 更新房源
```javascript
// 更新房源信息
const updateProperty = async (id, data) => {
  const response = await fetch(`/api/properties/${id}`, {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data)
  });
  
  return await response.json();
};
```

### 5. 导出Excel
```javascript
// 导出Excel
const exportExcel = async (region) => {
  const response = await fetch('/api/export/excel', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ region })
  });
  
  const result = await response.json();
  
  // 下载文件
  const link = document.createElement('a');
  link.href = result.data.downloadUrl;
  link.download = result.data.fileName;
  link.click();
};
```

## 附录：状态码说明

### HTTP状态码
- `200 OK`：请求成功
- `201 Created`：创建成功
- `400 Bad Request`：请求参数错误
- `404 Not Found`：资源不存在
- `500 Internal Server Error`：服务器错误

### 业务状态码
所有业务状态码定义在响应体的`error.code`字段中，具体见各接口的错误码说明。
